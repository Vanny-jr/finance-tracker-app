# Finance Tracker App

A full stack MERN application for tracking personal finances.

## Tech Stack

- MongoDB — Database
- Express.js — Backend API
- React.js — Frontend
- Node.js — Runtime

## Features

- Add expenses
- View all expenses
- Edit existing expenses
- Delete expenses

## How To Run

### Backend

cd backend
npm install
npm start

### Frontend

cd frontend
npm install
npm run dev
